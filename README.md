# VidaSalud

La empresa VidaSalud que se enfoca en el desarrollo de aplicaciones móviles de salud y bienestar integral ofreciendo servicios como planes de nutrición, programas de ejercicio y sesiones de mindfulness 
ha decidido lanzar una aplicación móvil innovadora que permitirá a los usuarios monitorear su salud, 
recibir recomendaciones personalizadas y conectarse con una comunidad de personas con intereses similares.

VidaSalud se encontró con diversos problemas al momento de el desarrollo de la aplicación, como la arquitectura, diseño, plataforma de desarrollo para tener una aplicación escalable y fácil de mantener con el tiempo
y a la vez proporcionar una experiencia óptima para el usuario.
